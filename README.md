## GraphBitStreamer - поиск несжатой графики в ресурсах игр

Автор программы: **Ilya "Steel Rat" Pavlovets**

Что она может:
+ представлять графически данные
+ каждый пискел может быть от 1 до 56 бит шириной (максимальное отображение всё равно 24 бита только)
+ лёгкое редактирование, по одной точке, чего вполне достаточно для правки
+ экспорт/импорт в BMP формате (убил бы его разработчиков, но он самый распространённый из несжатых, вроде)
+ экспорт RAW данных, может быть полезным для анализа сжатий типа LZ
+ запись/загрузка текущего проекта
+ в текущем проекте запоминает/восстанавливает 10 состояний по нажатию Ctrl/Shift 0-9

Что она не может:
- работать с файлами больше 260 Мб (слайдеры используют int), а больше и вряд ли надо
